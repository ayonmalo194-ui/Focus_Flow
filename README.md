# Focus_Flow
FocusFlow is an AI-powered productivity assistant designed to help users reduce distractions and improve deep focus during study or work. Using adaptive focus sessions, personalized analytics, and intelligent AI suggestions, it enables users to optimize their productivity habits. FocusFlow integrates Firebase for backend services.
